## Installing

0. Install maven. 
1. Clone this repository.
2. Navigate to the installation folder and run: 

``` mvn clean install ```

4. From the installation folder navigate to `target/nulab-0.0.1/`
5. Run the program using one of the available scripts.

Requirements:
Java 8 or above.
